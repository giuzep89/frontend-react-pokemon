# PokemonFetcher

A React application that fetches and displays Pokemon data from the PokeAPI in a paginated card layout.

## Tech Stack

- React 18
- Vite
- Axios

## Getting Started

Install dependencies:

```shell
npm install
```

Run the development server:

```shell
npm run dev
```

Open http://localhost:5173/ in your browser.
